# EserciziArduino
Sono presenti i quattro esercizi assegnati in classe sulla programmazione di arduino con thinkercad.
Esercizio 1.
Far lampaggeiare un led collegato ad uno dei pin di arduino.
Esercizio 2.
Implementare un semaforo utilizzando tre led.
Esercizio 3.
Utilizzare un pulsante per accendere e spegnere il semaforo.
Esercizio 4.
Simulare un incrocio con quattro semafori.
